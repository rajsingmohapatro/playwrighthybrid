## Download and Install Locally

You can either **clone** the repository or **download** it as a ZIP archive.

### Option 1: Clone with Git

```bash
git clone https://github.com/rajsingmohapatro/playwrighthybrid.git
cd playwrighthybrid
npm install
```

### Option 2: Download ZIP

- Go to [https://github.com/rajsingmohapatro/playwrighthybrid](https://github.com/rajsingmohapatro/playwrighthybrid)
- Click the green **Code** button, then **Download ZIP**
- Unzip and run:
  
```bash
cd playwrighthybrid
npm install
```
